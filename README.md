# Proyecto Login SQL

Sistema de autenticación conectado a una base de datos SQL Server (pubs).

## Instrucciones de uso
Para probar el acceso al sistema, puede utilizar los siguientes usuarios registrados en la tabla `employee`:

| Usuario (emp_id) | Apellido (Contraseña) |
| :--- | :--- |
| A-C71970F | Cruz |
| VPA30890F | Accorti |
| H-B39754M | Bennett |

## Requisitos
- SQL Server (LocalDB) instalado.
- Base de datos `pubs` configurada con el script `instpubs.sql`.
- .NET Desktop Runtime.
